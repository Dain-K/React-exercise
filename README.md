
#### yarn
- yarn 설치하기
```
$ npm install --g yarn
$ npm install --global yarn
```
#### 프로젝트 작업 환경 구축
```
$ yarn init
```
#### 프로젝트 파일 생성
- 프로젝트 명: react-exercise
- [주의]: 프로젝트 명에 대문자가 있으면 안된다. 
```
$ yarn create react-app react-exercise
``` 
```
$ cd react-exercise
```
#### 웹 페이지 열기
```
$ yarn start
```
#### react-router-dom 설치
```
$ yarn add react-router-dom
```
#### 로그인 시 필요한 모듈
```
$ yarn add axios
```
